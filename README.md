##IllagersVZombies is a Minecraft plugin that modifies Illagers, Piglin, and Zombie behaviour.
Designed for Project: Kompass.

This plugin modifies the following mob behaviours:

### Illagers:
- Will engage and attack all Zombie variants.
- Will engage and attack all Piglin variants.
- Will avoid any nearby Wardens.
- Witches will attack Zombies with healing potions.

### Zombies
- Will engage and attack all Illager raiders.
- Will engage and attack all Piglin variants.

### Piglins
- Will avoid all Zombie variants.
- Will only attack Illagers unless provoked.
- Will avoid any nearby Wardens.
